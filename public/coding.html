Setting Everything up

So, you want to make a basic website... Well you will ne Visual Studio Code installed with the Live Server extension. There is a dowload link on the home/main page and you can download it by clicking the words that say "Web Development". Then just simply choose your operatin system and you can watch the video below for a quick setup. Make sure on the setup screen to allow acess

My First Website

When you open Your Visual studio Code it should look something like this.

You will not have a tag called "Create a java project" because I have a JDK Installed which you probably wont have

You will now need to click the "Open folder button and find a folder that you will make your website in". Once you find a folder, open it and click on "Select Folder"

Now click on this logo

Make a file that looks like this and hit enter

Type a ! mark and hit enter. Something like this should appear.

Change the title tag to the title of your website and that's it! We will now start making the website!
